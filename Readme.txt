App that allows you to organize and view important stats on guarantees along with their clients, guarantors, Agencies, and many more, 


The credentials files contain all of the usernames and passwords needed to access the application.


The database is stored in an sql file called sggaranties.sql
to test you will need to change the settings in the app_local.php(data sources(default)) file in the config folder 
then php bin/cake.php server in the terminal

it's a CakePHP application 



features 
audit logging, stores every single change along with the time and user implicated 
user registration and login system
data visualization 
User-friendly interface 
adding, editing, deleting, and restoring
admin privileges
